Canvas, wat is het?

Dashboard

Wanneer je de Canvas website opent kom je gelijk bij Dashboard. Wat je hier ziet zijn tegels met namen, die namen zijn jouw klassen. Als je op zo'n tegel klikt kom je bij de lessen, daar staan opdrachten, uitleg en wat je de volgende les gaat doen.


Cursussen

Cursussen zijn de lessen die je hebt, er is een speciaal tabblad ervoor, maar je kan eigenlijk dus ook gewoon Dashboard gebruiken. Wanneer je bij een cursus pagina ben zie je aan de zijkant heel veel dingen.

Opdrachten

Bij opdrachten zie je... nou... opdrachten. Zo heb je dus een goed overzicht van de dingen die je nog moet doen. Klik op een opdracht en dan kan je het inleveren, scroll daarvoor heemaal naar beneden en sleep/selecteer het bestand dat je wilt inleveren. Zodra je het heb ingeleverd kan de docent punten geven en een opmerking plaatsen.


Kalender

In de kalender kan je zien op welke datum opdrachten ingeleverd moeten worden. Je kan daar zelf ook dingen inzetten, als je bijvoorbeeld een afspraak heb om aan een project te werken.


Inbox

Met inbox kan je berichten sturen per cursus naar klasgenoten. Dit is vooral handig als je vragen hebt. Dan kan je gewoon aan een vriend via canvas vragen en dat de cursus erbij zetten zodat ze weten om welk vak het gaat.
